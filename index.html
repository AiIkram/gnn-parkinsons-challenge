üöÄ Getting Started

Step 1: Fork Repository

Clone the repository and set up your local environment with Python 3.8+

Step 2: Install Dependencies

Install DGL, PyTorch, and other required packages from requirements.txt

Step 3: Generate Data

Run the data generation script to create train and test graph files

Step 4: Train Model

Step 5: Submit Predictions

Create a Pull Request with your submission CSV for automatic scoring
